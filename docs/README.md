# Dokumen Acuan

Letakkan dokumen konteks resmi Anda di folder ini:

- `BrandContext.md` — identitas merek, palet warna, tipografi, komponen, pola status.
- `ProductContext.md` — modul, peran pengguna, aturan privasi, urutan pengembangan.
- `struktur-laravel.md` — peta folder Laravel untuk tahap coding.

Dokumen ini menjadi sumber kebenaran. Setiap halaman dan modul mengacu ke sini.
